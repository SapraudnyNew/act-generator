## Active Projects
act-generator — генерация актов сдачи-приёмки из PDF счёт-договоров. Протестировано на счетах 1128–1132. Шаблон и правила стабильны.

## Key Rules
- Дата акта = дата окончания услуг из счёта (не дата договора)
- `payment_link` — запрашивать для каждого счёта отдельно
- `signatory` — родительный падеж (шапка), `signatory_short` — именительный Фамилия И.О. (подпись)
- `authority` — «Устава» по умолчанию, «Доверенности № X от ДД.ММ.ГГГГ г.» если доверенность
- Для ИП: `authority` = пустая строка, `kpp` = пустая строка — фраза «действующего на основании» убирается автоматически
- Если банковских реквизитов нет в счёте — запросить карточку предприятия
- Имя файла: `Акт к Счет-договору № {N} от {date}г.docx`
